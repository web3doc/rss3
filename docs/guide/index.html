简介

RSS3 是下一代提要标准，旨在支持高效和分散的信息分发。

我们已经看到了 Web1 时代，每个人都乐于将 RSS 用作联合的标准格式，而 Web2 时代，RSS 衰落并创建了平台之间的信息封闭。

我们都同意在 Web3 时代信息应该更加自由地分发，为了促进这一点，我们需要一种我们都可以用来联合信息的通用格式，从而降低使用和开发成本。所以我们创建了一个新的协议RSS3，RSS代表对历史的致敬，3代表Web3。

RSS3 是为统一信息格式和促进信息自由流动而创建的协议。可以在 此处 找到 RSS3 协议的详细信息。

对协议有基本的了解后，最快的上手方式是使用 SDK，或者直接请求 API。

阅读 我们的白皮书 也是了解更多关于协议和即将到来的去中心化网络的好方法。

如果您有任何问题，可以在 GitHub Discord Telegram 或 Twitter。我们很乐意提供帮助！

RSS3 及其附属网络是开放且无需许可的 - 如果您将 RSS3 用于您的项目，如果您 让我们知道，我们将很高兴。如果您需要，我们将提供技术或操作帮助。
